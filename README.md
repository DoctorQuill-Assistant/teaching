# Teaching

Public GitHub Pages repository for DrQuill Nanobot learning lessons.

## Dashboard

The site root (`index.html`) is a generated dashboard listing all topics, lessons, and references.
Each topic also gets its own generated landing page at `<topic>/index.html`.

To make content appear there:
- add lesson files under `<topic>/lessons/`
- add reference files under `<topic>/reference/`

Then deploy/push the site.
