French Lesson 0001: First polite French for ordering dinner

This lesson teaches one small, useful skill: how to begin a restaurant interaction politely and order one item with confidence.

Your win for today

By the end of this lesson, you should be able to say:

Bonjour. Je voudrais un steak-frites, s’il vous plaît.

Hello. I would like a steak and fries, please.

The 3 phrases to learn

  1. Bonjour — hello / good day. In French, starting with bonjour is especially important when beginning an interaction. Source.
  2. Je voudrais… — I would like… A polite way to order. Source.
  3. S’il vous plaît — please.

Mini pattern

Bonjour. Je voudrais [food], s’il vous plaît.

Examples:

  • Bonjour. Je voudrais un café, s’il vous plaît.
  • Bonjour. Je voudrais une soupe, s’il vous plaît.
  • Bonjour. Je voudrais un steak-frites, s’il vous plaît.

Quick practice

Type the missing polite opening word:

_____ . Je voudrais un café, s’il vous plaît.

Real-world note

Use bonjour when starting the interaction. If it is evening, bonsoir is also used. To wish someone a good day when leaving, say bonne journée rather than bonjour. Source.

Next useful phrase

When you are ready to pay: L’addition, s’il vous plaît.
