# Order dinner in French

Goal: help the user handle a simple dinner order in French with confidence.

## Core pattern

Use this polite structure:

- **Je voudrais...** = I would like...
- **Je vais prendre...** = I'll have...
- **Pour moi, ce sera...** = For me, it'll be...

For beginner use, **je voudrais** is the safest and most polite default.

## Essential phrases

### Arriving

- **Bonsoir.** = Good evening.
- **Une table pour deux, s'il vous plaît.** = A table for two, please.
- **On a une réservation au nom de Quill.** = We have a reservation under the name Quill.
- **Est-ce qu'on peut manger ici ?** = Can we eat here?

### Looking at the menu

- **La carte, s'il vous plaît.** = The menu, please.
- **Qu'est-ce que vous recommandez ?** = What do you recommend?
- **Qu'est-ce que c'est ?** = What is that?
- **Je ne mange pas de viande.** = I don't eat meat.
- **Je suis allergique aux noix.** = I'm allergic to nuts.

Note: in French restaurant language, **la carte** usually means the menu. **Le menu** often means a set meal or fixed-price option.

### Ordering drinks

- **Je voudrais un verre d'eau.** = I'd like a glass of water.
- **Je voudrais une carafe d'eau.** = I'd like a jug of tap water.
- **Je vais prendre un café.** = I'll have a coffee.
- **Pour boire, je voudrais un thé.** = To drink, I'd like a tea.

### Ordering dinner

- **Je voudrais le poulet, s'il vous plaît.** = I'd like the chicken, please.
- **Je vais prendre le poisson.** = I'll have the fish.
- **Pour moi, ce sera les pâtes.** = For me, it'll be the pasta.
- **Sans oignons, s'il vous plaît.** = Without onions, please.
- **Avec une salade.** = With a salad.

### During the meal

- **Excusez-moi.** = Excuse me.
- **Je voudrais encore du pain, s'il vous plaît.** = I'd like some more bread, please.
- **On peut avoir de l'eau ?** = Can we have some water?
- **C'est très bon.** = It's very good.

### Paying

- **L'addition, s'il vous plaît.** = The bill, please.
- **Je peux payer par carte ?** = Can I pay by card?
- **On partage l'addition ?** = Shall we split the bill?
- **Merci, bonne soirée.** = Thanks, have a good evening.

## Mini grammar you actually need

### 1. Je voudrais

**Je voudrais** is the most useful structure for polite ordering.

- **Je voudrais une soupe.** = I'd like a soup.
- **Je voudrais de l'eau.** = I'd like some water.

### 2. Articles matter

Use these common patterns:

- **un** for many masculine nouns: **un café**
- **une** for many feminine nouns: **une salade**
- **le / la / les** when choosing a specific dish from the menu: **je vais prendre le saumon**
- **du / de la / de l'** for some quantity of food or drink: **du pain, de la soupe, de l'eau**

### 3. Negatives for restrictions

- **Je ne mange pas de porc.** = I don't eat pork.
- **Je ne peux pas manger de gluten.** = I can't eat gluten.

After **ne ... pas**, **de** is common before food nouns.

## Survival dialogue

**Serveur :** Bonsoir.
**Vous :** Bonsoir. Une table pour une personne, s'il vous plaît.
**Serveur :** Bien sûr. Voilà la carte.
**Vous :** Merci. Qu'est-ce que vous recommandez ?
**Serveur :** Le poulet rôti est très bon.
**Vous :** D'accord. Je vais prendre le poulet rôti et une salade.
**Serveur :** Et à boire ?
**Vous :** Je voudrais une carafe d'eau, s'il vous plaît.
**Serveur :** Très bien.
...
**Vous :** L'addition, s'il vous plaît.
**Serveur :** Bien sûr.
**Vous :** Merci, bonne soirée.

## Fast pronunciation help

- **voudrais** sounds roughly like "voo-dray"
- **addition** sounds roughly like "ah-dee-see-ohn"
- **carte** sounds like "kart"
- **eau** sounds like "oh"
- **s'il vous plaît** sounds roughly like "seel voo pleh"

These are approximations for confidence, not perfect phonetics.

## What to memorize first

If the user only learns five things, learn these:

1. **Bonsoir.**
2. **Je voudrais...**
3. **Je vais prendre...**
4. **Je suis allergique à...**
5. **L'addition, s'il vous plaît.**

1. I'd like a table for two.
2. I'll have the fish.
3. I'm allergic to nuts.
4. The bill, please.
5. Can I pay by card?

1. **Je voudrais une table pour deux.**
2. **Je vais prendre le poisson.**
3. **Je suis allergique aux noix.**
4. **L'addition, s'il vous plaît.**
5. **Je peux payer par carte ?**
