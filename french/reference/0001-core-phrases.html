French Reference 0001: Core polite phrases

This quick reference supports the mission of ordering dinner politely in French.

Bonjour
Hello / good day. Use this first when starting an interaction.
Bonsoir
Good evening. Often used from around 6pm onward.
S’il vous plaît
Please.
Merci
Thank you.
Je voudrais…
I would like…
Je vais prendre…
I’ll have…
L’addition, s’il vous plaît.
The bill, please.

Sources: Lawless French — Greetings, Lawless French — Restaurant.
