# Learning Record 0002: Quiz-first and pronunciation preferences

## Context
The user said they learn best with quizzes and would like future lessons to help with pronunciation.

## Decision
The French topic should use a quiz-forward lesson structure. Each lesson should include active recall plus pronunciation support for key phrases.

## Implementation guidance
- Include at least one quiz block in every lesson.
- Prefer short formats: fill-in-the-blank, translation, and multiple choice.
- Add IPA when useful.
- Add an English-friendly pronunciation hint when useful.
- Add short notes for tricky French sounds.
