# Hessisch mission

Create a learner-friendly Hessisch teaching track focused on everyday Südhessisch from the Wiesbaden area, practical speaking confidence, humor, and usable regional sayings.

Primary user goals:
- learn to speak beginner-friendly Südhessisch
- use Hessisch more naturally in relaxed everyday situations
- understand and use common humorous expressions and sayings
- build confidence with pronunciation and rhythm, not just vocabulary

Lesson design requirements:
- each lesson should teach immediately usable spoken Hessisch
- each lesson should be short and include exercises
- each lesson should include practical idioms or regional expressions when useful
- each lesson should include simple pronunciation hints written for learners
- each lesson should explain contrasts with standard German clearly
- each lesson should include at least one mini dialogue
- each lesson should include quiz sections focused on active recall
- each lesson should include a self-check answer key
- each lesson should invite follow-up pronunciation questions

Quality note:
- prefer common, natural, alltagsnah Südhessisch forms over exaggerated stage-dialect spellings
- when forms vary regionally, favor Wiesbaden-area usage or label the variation briefly
