# Portuguese lesson 2: Ordering dinner

Goal: help the user order a simple dinner in Portuguese with confidence.

## Core survival phrases

Use these first in a restaurant:

- **Boa noite.** = Good evening.
- **Uma mesa para um, por favor.** = A table for one, please.
- **Uma mesa para dois, por favor.** = A table for two, please.
- **Posso ver o menu, por favor?** = Can I see the menu, please?
- **Queria pedir.** = I would like to order.
- **Para beber, queria água.** = To drink, I’d like water.
- **Para começar, queria sopa.** = To start, I’d like soup.
- **Como prato principal, queria frango.** = As a main course, I’d like chicken.
- **A conta, por favor.** = The bill, please.

A very safe beginner pattern is **Queria...** = “I would like ...”

## Useful restaurant vocabulary

- **o menu / a ementa** = the menu
- **a mesa** = the table
- **a água** = water
- **o pão** = bread
- **a sopa** = soup
- **o peixe** = fish
- **a carne** = meat
- **o frango** = chicken
- **a sobremesa** = dessert
- **a conta** = the bill
- **o empregado / a empregada** = waiter / waitress

## Ordering politely

These are high-frequency beginner phrases:

- **Queria água, por favor.** = I would like water, please.
- **Queria o peixe.** = I would like the fish.
- **Queria uma sopa e uma salada.** = I would like a soup and a salad.
- **Sem carne, por favor.** = Without meat, please.
- **Com arroz, por favor.** = With rice, please.

## Asking simple questions

- **O que recomenda?** = What do you recommend?
- **Tem opção vegetariana?** = Do you have a vegetarian option?
- **Tem menu em inglês?** = Do you have a menu in English?
- **Isto é picante?** = Is this spicy?
- **Pode repetir, por favor?** = Can you repeat, please?

## Asking for the bill

The most important phrase at the end:

- **A conta, por favor.** = The bill, please.

You may also hear:

- **Como vai pagar?** = How will you pay?
- **Dinheiro ou cartão?** = Cash or card?

Useful answers:

- **Em dinheiro.** = In cash.
- **Com cartão.** = By card.

## Mini dialogue

**Empregado:** Boa noite.
**Cliente:** Boa noite. Uma mesa para um, por favor.
**Empregado:** Claro.
**Cliente:** Posso ver o menu, por favor?
**Empregado:** Sim.
**Cliente:** Queria água e sopa. Como prato principal, queria frango.
**Empregado:** Muito bem.
**Cliente:** A conta, por favor.

## Fast pronunciation help

- **queria** ≈ "keh-REE-ah"
- **menu** ≈ "meh-NOO"
- **ementa** ≈ "eh-MEN-tah"
- **conta** ≈ "KOHN-tah"
- **frango** ≈ "FRAHN-goo"
- **peixe** ≈ "PAY-shuh"

These are confidence approximations, not exact phonetics.

## What to memorize first

If the user only memorizes six restaurant phrases, learn these:

1. **Uma mesa para dois, por favor.**
2. **Posso ver o menu, por favor?**
3. **Queria água, por favor.**
4. **Queria o peixe / frango.**
5. **O que recomenda?**
6. **A conta, por favor.**
