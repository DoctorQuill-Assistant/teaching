Breakfast and simple orders

In this lesson, you will practice tiny real-life breakfast orders in Portuguese. The goal is not lots of new vocabulary. The goal is to combine things you already know with a few useful new words.

Focus: short café and breakfast phrases you can say out loud with confidence.

New vocabulary in this lesson: only 4 items. We heavily reuse Queria..., por favor, numbers, and café phrases from earlier lessons.

Practical objective

  • Order a simple breakfast item and a drink.
  • Use Para mim... naturally.
  • Say one small complete order without freezing.

Today’s 4 focus items

Para mim... = For me...
torrada = toast
manteiga = butter
sumo de laranja = orange juice

That is enough for today. Everything else is mostly review.

The key beginner pattern

Para mim, uma água. = For me, a water.
Para mim, um café. = For me, a coffee.
Para mim, uma torrada. = For me, a toast.

Para mim... is very useful when ordering for yourself, especially in a group or in a simple café interaction.

Core phrases

Queria um café, por favor. = I would like a coffee, please.
Para mim, uma torrada. = For me, a toast.
Para mim, um sumo de laranja. = For me, an orange juice.
Queria uma torrada com manteiga. = I would like toast with butter.
E para beber? = And to drink?
É tudo, obrigado/a. = That’s all, thank you.

Useful breakfast words

  • café — coffee
  • água — water
  • sumo de laranja — orange juice
  • torrada — toast
  • manteiga — butter
  • pastel de nata — custard tart

Mini dialogue: simple breakfast order

Server: Bom dia.

You: Bom dia. Queria um café, por favor.

Server: E para comer?

You: Para mim, uma torrada com manteiga.

Server: Mais alguma coisa?

You: É tudo, obrigado/a.

Mini dialogue: drink plus food

You: Com licença.

Server: Sim?

You: Para mim, um sumo de laranja e uma torrada, por favor.

Server: Claro.

Pronunciation confidence hints

  • para mim ≈ "PAH-rah meeng"
  • torrada ≈ "toh-HAH-dah"
  • manteiga ≈ "mahn-TAY-gah"
  • sumo de laranja ≈ "SOO-moo d luh-RAN-zhah"
  • para beber ≈ "PAH-rah beh-BEHR"

These are simple learner-friendly hints for confidence, not exact phonetics.

Small pattern note

You already know Queria.... Now add Para mim... as another easy ordering pattern. You do not need to choose one forever. Both are useful.

For today, focus on saying one full breakfast order smoothly, not on memorizing lots of food words.

What to memorize first

  1. Para mim...
  2. uma torrada
  3. sumo de laranja
  4. Queria um café, por favor.
  5. É tudo, obrigado/a.

Quiz 2: Fill in the blank

  2. uma _____ (toast)
  3. com _____ (butter)
  4. sumo de _____ (orange)

Quiz 3: Mini response drills

Answer in Portuguese:

  1. The server asks: E para beber? You say: “For me, a coffee.”
  2. You want to say: “For me, an orange juice.”
  3. You want to say: “I would like toast with butter.”
  4. The server asks: Mais alguma coisa? You say: “That’s all, thank you.”

Quiz 4: Situations

Say the Portuguese phrase you would use:

  1. You want to order a coffee and then add toast for yourself.
  2. You want to order orange juice for yourself.
  3. You want to say “toast with butter.”
  4. You want to finish the order politely.

Speaking drill

Say these aloud three times:

  1. Para mim, uma torrada.
  2. Para mim, um sumo de laranja.
  3. Queria um café, por favor.
  4. Queria uma torrada com manteiga.
  5. É tudo, obrigado/a.

Self-check answer key

  • For me... → Para mim...
  • toast → torrada
  • butter → manteiga
  • orange juice → sumo de laranja
  • For me, a coffee. → Para mim, um café.
  • I would like toast with butter. → Queria uma torrada com manteiga.
  • That’s all, thank you. → É tudo, obrigado/a.
