Portuguese Reference: Breakfast and Simple Orders

Quick review sheet for tiny breakfast and café orders in beginner Portuguese.

High-value phrases

  • Para mim... — For me...
  • Para mim, uma torrada. — For me, a toast.
  • Para mim, um sumo de laranja. — For me, an orange juice.
  • Queria uma torrada com manteiga. — I would like toast with butter.
  • E para beber? — And to drink?
  • É tudo, obrigado/a. — That’s all, thank you.

Useful words

  • torrada — toast
  • manteiga — butter
  • sumo de laranja — orange juice
  • café — coffee

Pronunciation confidence hints

  • para mim ≈ "PAH-rah meeng"
  • torrada ≈ "toh-HAH-dah"
  • manteiga ≈ "mahn-TAY-gah"
  • sumo de laranja ≈ "SOO-moo d luh-RAN-zhah"

Approximate help for confidence, not exact phonetics.

Mini self-check

  • for me → para mim
  • toast with butter → torrada com manteiga
  • for me, an orange juice → para mim, um sumo de laranja
