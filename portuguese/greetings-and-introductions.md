# Portuguese lesson 1: Greetings and introductions

Goal: help the user start a simple conversation politely in Portuguese.

## The most important greetings

Use these first:

- **Olá.** = Hello.
- **Bom dia.** = Good morning.
- **Boa tarde.** = Good afternoon.
- **Boa noite.** = Good evening / good night.

A safe beginner habit: start with **olá** or the time-based greeting.

When leaving:

- **Adeus.** = Goodbye.
- **Até logo.** = See you later.
- **Até amanhã.** = See you tomorrow.

## Polite social phrases

- **Por favor.** = Please.
- **Obrigado / Obrigada.** = Thank you.
- **De nada.** = You're welcome.
- **Desculpe.** = Sorry / excuse me.
- **Com licença.** = Excuse me.

Use **obrigado** if you are male and **obrigada** if you are female.

## Ask and answer "How are you?"

- **Tudo bem?** = How are you? / Everything good?
- **Tudo bem.** = I'm fine.
- **Tudo bem, e consigo?** = I'm fine, and you? (polite)
- **Tudo bem, e contigo?** = I'm fine, and you? (informal)

For beginner use with strangers, **Tudo bem?** is the easiest default.

## Introduce yourself

- **Chamo-me Quill.** = My name is Quill.
- **O meu nome é Quill.** = My name is Quill.
- **Sou americano / americana.** = I am American.
- **Falo um pouco de português.** = I speak a little Portuguese.
- **Sou principiante.** = I am a beginner.

For beginner use, **O meu nome é...** is very easy and clear.

## Ask someone their name

- **Como se chama?** = What is your name? (polite)
- **Como te chamas?** = What's your name? (informal)

For beginner use, prefer **Como se chama?** with strangers.

## Mini dialogue

**A:** Olá. Bom dia.
**B:** Bom dia.
**A:** O meu nome é Quill. E você?
**B:** Chamo-me Ana.
**A:** Muito prazer.
**B:** Igualmente.

## Fast pronunciation help

- **olá** ≈ "oh-LAH"
- **bom dia** ≈ "bohn DEE-ah"
- **boa tarde** ≈ "BOH-ah TAR-jee" or "TAR-d(uh)"
- **boa noite** ≈ "BOH-ah NOY-t(uh)"
- **obrigado** ≈ "oh-bree-GAH-doo"
- **desculpe** ≈ "desh-KOOL-peh"

These are confidence approximations, not exact phonetics.

## What to memorize first

If the user only learns five things, learn these:

1. **Olá.**
2. **Bom dia.**
3. **Tudo bem?**
4. **O meu nome é...**
5. **Obrigado / Obrigada.**

## Quick practice

Translate into Portuguese:

1. Hello.
2. Good afternoon.
3. My name is Quill.
4. I speak a little Portuguese.
5. Thank you.

Answers:

1. **Olá.**
2. **Boa tarde.**
3. **O meu nome é Quill.**
4. **Falo um pouco de português.**
5. **Obrigado / Obrigada.**
