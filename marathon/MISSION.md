# Marathon mission

Create a learner-friendly marathon coaching track focused on helping the user complete a marathon in one year, finish comfortably, and be able to say he has done a marathon.

Primary user goals:
- complete a marathon in one year
- finish comfortably rather than chase an aggressive time goal
- build sustainable training habits around 3 to 4 runs per week
- use short, practical teaching that supports real training decisions

Training constraints and preferences:
- realistic running frequency: 3 to 4 days per week
- weekends plus Monday and Friday are easiest for running
- middle of the week is hardest
- no current pain and no doctor restrictions
- no gym access desired; strength work should be home-based
- some home weights are available
- likely warm race conditions
- watch-based wrist heart rate available from Garmin

Lesson design requirements:
- each lesson should have a short practical objective
- each lesson should explain a small number of useful concepts clearly
- each lesson should include actionable training guidance
- each lesson should include a mini scenario or example week when useful
- each lesson should include a quiz or self-check section
- each lesson should encourage follow-up questions

Quality note:
- prioritize safe, sustainable progression and practical coaching over hype
- use effort-first guidance, with pace and heart rate as secondary tools when needed
