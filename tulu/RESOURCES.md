# Tulu resources

Core public resources for this topic:
- Easy Tulu lessons index: https://www.easytulu.com/p/tulu-lessons.html
- Easy Tulu useful phrases: https://www.easytulu.com/2016/12/tulu-lesson-41-useful-phrases-in-tulu.html
- Easy Tulu subjective pronouns: https://www.easytulu.com/2016/01/tulu-lesson-2-subjective-pronouns.html

Usage guidance:
- treat online Tulu resources as helpful but inconsistent references
- cross-check high-value beginner phrases before baking them into lessons when possible
- use learner-friendly romanized forms and lightweight pronunciation approximations
