# Tulu mission

Create a learner-friendly Tulu teaching track focused on speaking, listening support through pronunciation approximations, and practical everyday communication.

Primary user goals:
- learn to speak beginner Tulu
- build confidence pronouncing common words and phrases
- learn through short, practical, quiz-first lessons
- rely on scarce but useful public resources carefully, since Tulu has fewer learning materials than larger languages

Lesson design requirements:
- each lesson should teach immediately usable spoken Tulu
- each lesson should include simple pronunciation hints written for English-speaking learners
- each lesson should include a short practical objective
- each lesson should include core phrases with English meanings
- each lesson should include at least one mini dialogue
- each lesson should include quiz sections focused on active recall
- each lesson should include a self-check answer key
- each lesson should invite follow-up pronunciation questions

Quality note:
- because Tulu resources can vary in spelling and transliteration, prefer clearly sourced beginner forms and note when a pronunciation hint is approximate rather than exact
